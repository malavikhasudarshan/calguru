# CalGuru Color Palette

## Official Color Scheme

The CalGuru application uses a warm, earthy, and vibrant color palette inspired by the landing page animation. This palette replaces the previous Berkeley Blue and Gold scheme with a more diverse and playful set of colors.

### Primary Colors

| Color Name | Hex Code | Usage |
|------------|----------|-------|
| **Dusty Rose** | `#C97B84` | Accents, icons, gradients |
| **Gold** | `#D4A017` | Highlights, badges, active states |
| **Brown** | `#6B3E26` | Text accents, gradients |
| **Terracotta** | `#C65D3B` | Primary buttons, CTAs, headings |
| **Purple** | `#8C7AA9` | Interactive elements, filters, active states |
| **Dark Green** | `#2F4F3F` | Headers, primary text, navigation |
| **Cream** | `#F2E8DC` | Backgrounds (with paper texture) |

### Background Colors

- **Cork Board**: `#a67c52` with texture image
- **Paper Texture**: `#faf8f5` with SVG noise overlay

### Color Applications

#### Navigation & Headers
- Main header background: **Dark Green** (`#2F4F3F`)
- Active navigation borders: **Gold** (`#D4A017`)
- Header text: White

#### Buttons & CTAs
- Primary buttons: **Terracotta** (`#C65D3B`)
- Secondary buttons: **Dark Green** (`#2F4F3F`) outline
- Active/selected states: **Purple** (`#8C7AA9`)
- Bookmark/highlight: **Gold** (`#D4A017`)

#### Interactive Elements
- Category filters (active): **Purple** (`#8C7AA9`)
- Hover borders: **Purple** (`#8C7AA9`)
- Focus rings: **Purple** with 10% opacity
- Links: **Dark Green** (`#2F4F3F`)

#### Text & Typography
- Primary headings: **Dark Green** (`#2F4F3F`)
- Secondary headings: **Terracotta** (`#C65D3B`)
- Body text: `#333333`
- Metadata/timestamps: `#666666`

#### Badges & Tags
- Category badges: **Purple** (`#8C7AA9`)
- Status badges: **Gold** (`#D4A017`)
- Visibility badges: Various (Friends: Dark Green, Berkeley: Gold)

#### Gradients
- Avatar gradients: **Dusty Rose** to **Purple**
- Profile cover: **Dark Green** to **Purple** to **Dusty Rose**
- Coming soon notices: **Terracotta** to **Dusty Rose**

### Design Principles

1. **Warm & Welcoming**: The palette uses warm earth tones to create a friendly, approachable feel
2. **Playful & Vibrant**: Multiple colors add visual interest without overwhelming
3. **Accessible**: All color combinations maintain WCAG contrast ratios for readability
4. **Consistent**: Each color has a specific purpose across all components
5. **Cork Board Aesthetic**: The background textures and colors evoke a physical bulletin board

### Component-Specific Usage

#### Inquiries (ASK)
- Sidebar filters: Purple when active
- Post button: Terracotta
- Top-level comments: Dusty Rose left border
- Replies: Gold left border

#### Pins (PHOTOBOX)
- Create pin button: Terracotta
- Visibility options: Purple when active
- Upload areas: Purple dashed border
- Filter buttons: Purple hover/active

#### Feed
- Category filters: Purple when active
- Action buttons: Purple for likes, Gold for bookmarks
- User avatars: Dusty Rose/Purple gradient
- Post categories: Gold badges

#### Profile
- Cover photo: Dark Green to Purple to Dusty Rose gradient
- Avatar: Terracotta to Dusty Rose gradient
- Tab buttons: Purple when active
- Circle cards: Purple hover state

### Migration from Previous Palette

**Old Colors → New Colors**
- Berkeley Blue (`#003262`) → Dark Green (`#2F4F3F`) or Purple (`#8C7AA9`)
- California Gold (`#FDB515`) → Gold (`#D4A017`)
- Blue gradients → Warm gradients (Terracotta, Dusty Rose, Brown)

This palette provides a cohesive, warm, and inviting visual identity that distinguishes CalGuru while maintaining excellent usability and accessibility.
